# 🎮 Go-Tic-Tac-Toe

A fun two-player Tic Tac Toe game built using HTML, CSS, and vanilla JavaScript — playable directly in the browser.

## ✨ Features

- Two-player gameplay (X and O)
- Real-time win detection across rows, columns, and diagonals
- Live turn indicator showing whose move it is
- Winning line animation
- Game over sound effect
- Reset button to restart instantly

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🕹️ How to Play

1. Open the game in your browser
2. Player X goes first
3. Players take turns clicking on empty cells
4. First player to get 3 in a row wins!
5. Click Reset to start a new game
